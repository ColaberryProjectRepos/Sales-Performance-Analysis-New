Creating Scatter Chart for Sales By Team 19


  1. 1. Scatter Chart:

    • From the Visualizations pane, select "Scatter Chart."

  2. 2. Build Visual Tab:

    • Values: Sales Team.
    • X-axis: Sales per Mth.
    • Y-Axis: Sales YoY%.
    • Size: Sales per Mth.
    • Tooltips: Sales per Mth, Sales YoY%, GPM%, GPM YoY%, Discount%, Discount YoY%.

  3. 3. Format Visual Tab - General:

    • Properties: Size=Height=299, Width=335, Position=Horizontal=505, Vertical=421.
    • Title=On, Text= Sales By Team, Font=DIN (14), Text color=#12239e, Background color=#ADFAE7, Horizontal alignment=Center.
    • Effects: Background=Off,
    • Visual boarder=On, Color=#e6e6e6
    • Tooltips=On, Font= Segoe UI (9), Color= #C8C8C8, Value color=#EAEAEA, Background color= #333333, Transparency= 5%.

  4. 4. Format Visual - Visual:

    • X-axis Values: Font=Segoe UI (9), Font Color=#605E5C, Title = On, Font= DIN(12), Color=#70bbff.

    • Y-axis Values: Font=Segoe UI Bold (9), Font color=#605E5C, Display Units=Auto, Title = On, Font= DIN(12), Color=#12239e.

    • Legend=On, Position= Top left, Text= Segoe UI (9), Color=#605E5C.

    • Gridlines: Horizontal=On, Style= Solid, Color=#F3F2F1. Vertical=On, Style= Dotted, Color=#C8C6C4.

    • Markers: Color Border=On, Size=-10. Conditional Formatting:

      • Format style= Gradient.
      • What field= GPM%.
      • Lowest value color= #118DFF.
      • Highest value color=#12239E.

    • Category label: On, Values, Font=Segoe UI (9), Font color=#605E5C, Background=Off.
