Creating Visual Cards For KPIs 14

  • Create Sales per Mth Visual Card:
  • 1. Click on an empty space on the canvas from the Visualizations pane, select "Visual Card."
  • 2. Drag the "Sales per Mth" measure to the "Fields" value box.
  • 3. General Properties:
    • Size: Height=133, Width=205.
    • Position: Horizontal=0, Vertical=221.
    • Title=On, Text=Sales per Mth, Font=DIN (14), Text color=#FFFFFF, Background color=#ADFAE7, Horizontal alignment=Center.
    • Effects: Background=Off, Visual border color=#e6e6e6
    • 4. Visual Callout Value:
    • Font=DIN(26), Font Color=70bbff
    • 5. Sales YoY%: Dag the Sales YoY% measure to the fields value box fill up figures provided below for the necessary fields
    • General Properties:

        • Size: Height=31, Width=72.
        • Position: Horizontal=133, Vertical=323.
        • Title=Off
        • Effects: Background=Off
        • 6. Visual Callout Value:
        • Font=Segoe ui Semibold(13), Font Color=#12239E
        • If Value > 0, and <= 'Max' Then Color = #079952
        • If Value >=Min, and < '0' Then Color = #C00000
        • If Value = 0 Color =#e6e6e6
