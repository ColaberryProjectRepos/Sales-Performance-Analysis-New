Creating Sales by Region Line and Cluster Column Chart 17


  1. Create Line and Clustered Column Chart:

    1. Click on an empty space on the canvas.

    • From the Visualizations pane, select "Line and Clustered Column Chart."

  2. 2. Build Visual Tab:

    • X-axis Values: Region,State.
    • Y-axis Column Values: Sales per Mth.
    • Line Values: Sales YoY%, GPM%, GPM YoY%, Discount%, Discount YoY%.
    • Tooltips: Sales YoY%, GPM%, GPM YoY%, Discount%, Discount YoY%.

3. Format Visual Tab:

General Properties: Size=Height=332, Width=581, Position=Horizontal=202, Vertical=89.

    • Title=On, Text= Sales By Year, Font=DIN (14), Text color=#12239E, Background color=#ADFAE7, Horizontal alignment=Center.
    • Effects: Background=OFF
    • Visual boarder=On, Color=#e6e6e6
    • Tooltips=On, Font= Segoe UI (9), Color= #C8C8C8, Value color=#EAEAEA, Background color= #333333, Transparency= 5%.

  2. 4. Format Visual:

    • X-axis Values: Font=Segoe UI (9), Font color=#C8C8C8, Title = Off.
    • Y-axis Values: Font=Segoe UI Bold (9), Font color=#605E5C, Display Units=Auto, Title = Off.
    • Secondary Y-axis =On.
    • Legend=On, Position= Top left, Match line color= On.
    • Gridlines: Horizontal=On, Style= Dotted, Color=#C8c6c4. Vertical=On, Style= Dotted, Color=#C8c6c4..
    • Columns color= #70bbff.
    • Lines Colors: Sales YoY% =#12239e, GPM%= #d9b300, GPM YoY% = #e66c37, Discount% = # E044A7, Discount YoY% = #744Ec2
    • Markers= Off.
    • Data labels: Off.
