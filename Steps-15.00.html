Creating KPI Arrow Indicator 15


  • Create KPI Sales YoY% Arrow Indicator Visual Card:
  • 1. Click on an empty space on the canvas.
    • From the Visualizations pane, select "Visual Card."
  • 2. General Properties:
    • Size: Height=34, Width=56
    • Position: Horizontal=111, Vertical=410.
    • Title=Off.
    • Effects: Background=Off.
  • 3. Visual Callout Value:
  • Font=DIN(34)
  • Color = Fx (Clink on the function tab and fill values)
  • If Value > 0, and <= 'Max' Then Color = #079952
  • If Value >=Min, and < '0' Then Color = #C00000
  • If Value = 0 Color =#e6e6e6
